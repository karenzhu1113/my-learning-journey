# my-learning-journey
学习仓库1.0

## 学习目标

**核心定位**：以6年B端产品经理的业务理解与需求拆解能力为根基，系统掌握AI辅助开发技能，实现从"需求定义者"到"需求落地者"的跨越。

### 短期目标（1-3个月）
- 熟练使用AI编程工具（如Claude Code、Cursor、Trae等）完成前端页面与基础后端接口的开发
- 掌握Prompt Engineering核心技巧，能与AI高效协作完成功能拆解、代码生成与调试
- 独立交付1-2个完整的MVP产品（如内部工具、效率类应用），跑通"想法→原型→上线"全流程

### 中期目标（3-6个月）
- 建立B端PM视角的技术判断力：能评估技术方案可行性、识别AI生成代码的风险点与边界
- 掌握Next.js + React全栈开发能力，理解数据库设计、API设计、部署流程
- 将AI辅助开发能力反哺PM工作：快速制作可交互原型验证需求，缩短产品验证周期

### 长期目标（6-12个月）
- 成为"懂技术的B端PM"：在产品规划中能更精准评估研发成本、把控技术债、与研发团队高效对话
- 探索AI Agent / 工作流自动化在B端业务场景的应用，打造差异化产品竞争力
- 沉淀一套"PM + AI辅助开发"的方法论，赋能团队与行业实践

### 衡量标准
- 能独立完成从需求到上线的全栈交付
- 产出的AI生成代码具备生产级质量（可维护、可扩展）
- 形成可复用的AI协作开发SOP
